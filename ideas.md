## Идея 1: «Глубокий космос» — Dark Tech Futurism

**Design Movement**: Cyberpunk Minimalism / Dark SaaS

**Core Principles**:
- Тёмный фон (#0A0E1A) с синими неоновыми акцентами
- Сетка из тонких линий как фоновая текстура (tech grid)
- Данные и цифры как главные визуальные элементы
- Контраст: тёмное пространство + яркие синие вспышки

**Color Philosophy**:
- Background: #0A0E1A (почти чёрный с синим оттенком)
- Primary: #1E9BF0 (бренд-синий)
- Accent: #00D4FF (неоновый голубой)
- Surface: rgba(30, 155, 240, 0.08) (стеклянные карточки)
- Text: #E8F4FF (холодный белый)
- Эмоция: технологичность, эксклюзивность, будущее

**Layout Paradigm**:
- Асимметричные секции с диагональными разделителями
- Hero: большой текст слева, анимированная 3D-сфера справа
- Калькулятор: тёмная панель с неоновыми слайдерами

**Signature Elements**:
- Анимированная сетка точек/линий на фоне (particle network)
- Glassmorphism карточки с синим свечением
- Неоновые подчёркивания у ключевых цифр

**Interaction Philosophy**:
- Hover: синее свечение на карточках
- Scroll: параллакс-движение фоновой сетки
- Калькулятор: анимация чисел при изменении

**Animation**:
- Entrance: fade-up с задержкой (stagger)
- Числа: counter animation при попадании в viewport
- Фон: медленно движущаяся сетка частиц
- CTA кнопка: пульсирующее свечение

**Typography System**:
- Headlines: Bebas Neue или Oswald (bold, condensed)
- Body: Manrope (modern, readable)
- Numbers: Roboto Mono (monospace для цифр)

## Идея 2: «Чистая технология» — Light Tech Editorial

**Design Movement**: Swiss Editorial + Tech Startup

**Core Principles**:
- Белый фон с синими акцентами (точно как в презентации)
- Редакционная типографика: крупные заголовки, чёткая иерархия
- Карточки с тенями и скруглёнными углами
- Числа и данные как визуальные якоря

**Color Philosophy**:
- Background: #FFFFFF / #F7F9FC
- Primary: #1E9BF0 (бренд-синий)
- Blue cards: #1E9BF0 (как в презентации)
- Lavender accent: #E8EEFF (светло-лавандовый)
- Dark text: #0D1117
- Эмоция: доверие, профессионализм, ясность

**Layout Paradigm**:
- Полноширинные секции с чередованием белого и светло-синего фона
- Hero: большой заголовок + синяя кнопка + 3D Telegram-иконка
- Горизонтальный скролл для карточек партнёрства
- Калькулятор: светлая панель с синими слайдерами

**Signature Elements**:
- Синие pill-бейджи с текстом (как в презентации)
- Большие декоративные числа на фоне (светло-серые)
- Синие заполненные карточки для ключевых данных

**Interaction Philosophy**:
- Hover: лёгкий подъём карточки (translateY -4px)
- Scroll: плавное появление блоков (AOS-style)
- Калькулятор: мгновенный пересчёт

**Animation**:
- Entrance: slide-up + fade при скролле
- Числа: animated counter
- Hero: floating 3D объект
- Кнопки: scale + shadow on hover

**Typography System**:
- Headlines: Montserrat ExtraBold / Black
- Body: Nunito Sans (дружелюбный, читаемый)
- Numbers: Montserrat Bold

## Идея 3: «Градиентный поток» — Glassmorphism Gradient

**Design Movement**: Modern Glassmorphism + Gradient Mesh

**Core Principles**:
- Белый фон с синими и лавандовыми градиентными пятнами
- Стеклянные карточки (backdrop-filter: blur)
- Плавные градиенты как фоновые акценты
- 3D-объекты как в презентации (стеклянный Telegram)

**Color Philosophy**:
- Background: #FFFFFF с синими/лавандовыми gradient blobs
- Primary: #1E9BF0
- Glass: rgba(255,255,255,0.7) + blur
- Gradient: от #E8F4FF до #D4D4FF
- Эмоция: современность, лёгкость, инновации

**Layout Paradigm**:
- Hero: полноэкранный с gradient mesh фоном и стеклянным 3D объектом
- Floating cards с glassmorphism эффектом
- Калькулятор: стеклянная панель на градиентном фоне
- Секции разделены плавными волнами

**Signature Elements**:
- Gradient mesh blobs в фоне секций
- Glassmorphism карточки с тонкой рамкой
- Floating 3D Telegram объект в hero

**Interaction Philosophy**:
- Hover: усиление blur + лёгкое свечение
- Scroll: параллакс для gradient blobs
- Калькулятор: анимированные числа

**Animation**:
- Hero: медленно вращающийся 3D объект (CSS)
- Gradient blobs: медленное движение (keyframes)
- Карточки: stagger fade-in при скролле
- Числа: counter animation

**Typography System**:
- Headlines: Raleway ExtraBold (элегантный, современный)
- Body: Inter (читаемый)
- Accent: Raleway Bold для цифр

## Выбранная концепция: Идея 3 — «Градиентный поток»

Glassmorphism + Gradient Mesh наилучшим образом соответствует визуальному стилю презентации LabTG (3D стеклянные объекты, синие/лавандовые тона, белый фон). Эта концепция создаёт ощущение технологичности и современности, сохраняя корпоративный профессионализм.
